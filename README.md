# Calculator (Python)

This is a simple command-line calculator built using Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles invalid input
- Prevents division by zero

## Technologies Used
- Python

## How to Run
1. Run the Python file
2. Enter two numbers
3. Choose operation (+, -, *, /)
4. Get the result
